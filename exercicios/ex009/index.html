Listas

Listas Ordenadas

Para usarmos a lista ordenada temos que usar a tag <ol>. Segue exemplo:

  1. Acordar
  2. Escovar dente
  3. Lavar roupa
  1. Tomar Banho
  2. Lavar Banheiro

Listas não ordenadas

Para usarmos a lista não ordenada temos que usar a tag <ul>. Segue Exemplo:

  • Feijão
  • Arroz
  • macarrão
  • Farinha

Listas de definições

Toda lista de definições está dentro de uma tag <dl> cada termo é um <dt> e cada descrição é um <dd>. Segue Exemplo:

HTML
Linguagem de marcação utilizada para criar o conteúdo de sites.
CSS
Linguagem de marcação para a especificação de estilos em sites.
JavaScript
Linguagem de programação para criar interatividades em sites.
